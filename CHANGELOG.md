# Changelog

## [1.0.0] - 2025-01-09

### Added
- Initial release
- Sound notifications for party chat messages
- Sound notifications for player joined/left events
- Sound notification for group disbanded
- 9 built-in WoW sounds
- LibSharedMedia-3.0 integration for custom sounds
- Configurable audio channel selection
- Per-event enable/disable toggles
- Sound test functionality
- Slash commands: `/pp` and `/partyping`
- Macro support: `/run PartyPing:Toggle()`
